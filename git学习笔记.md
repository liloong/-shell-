
## 常用命令

#### 首次拉取代码
git clone xxx

#### 拉取更新
git pull 

#### 查看分支
git branch -v

#### 创建本地dev分支
git branch dev

#### 切换到dev分支
git checkout dev

#### 提交单个文件
git add filename

#### 提交全部文件
git add .

#### 
git commit -m "内容"

#### 提交到远程仓库
git push

#### 提交到远程分支仓库
git origin 本地分支名：远程分支名

#### git checkout -b test 是git branch iss53 git checkout iss53这两条命令的简写
git checkout -b test
